# MyYoutube

> A simple youtube client using the youtube api.

## Run it

```bash
export YOUTUBE_API_KEY=<your api key>
export YOUTUBE_CHANNEL_ID=<the channel id>
```

Run:

```bash
bun start
```

## Development

Install Node:

```bash
asdf install nodejs 20.10.0
```

Install Bun:

```bash
npm i -g bun
```

Install dependencies:

```bash
bun install
```

Run:

```bash
bun run start
```
